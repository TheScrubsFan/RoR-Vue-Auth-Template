# Шаблон приложения RoR + Vue

## Description

Шаблон RoR-приложения с фронтом на Vue и аутентификацией посредством Devise.

Стек: 
- аутентификация - devise;
- фронтэнд - vue;

* Rails - 6.0.3.5
* Vue - 2.6.12
* Node - 10.13.0
