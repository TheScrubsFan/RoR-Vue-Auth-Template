# Шаблон приложения RoR + Vue

## Description

Шаблон RoR-приложения с фронтом на Vue и аутентификацией через JWT.

Стек:
- аутентификация - jwt;
- фронтэнд - vue;
- css фреймворк - vuetify;
- тестирование - rspec;

## Built With

* Ruby - 2.7.3
* Rails - 6.0.3.5
* Vue - 2.6.12
* Node - 10.13.0
